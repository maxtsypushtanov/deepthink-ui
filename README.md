# DeepThink

**Думай глубже. Решай точнее.**

Когнитивная платформа, в основе которой — **Нейрон**, ИИ-агент с 10 стратегиями мышления, памятью и характером. Дешёвая модель + умный движок рассуждений = результат уровня топовой модели.

## Философия

DeepThink — не чат-обёртка. Это платформа когнитивного усиления.

**Нейрон** — разум платформы. Он не просто отвечает на вопросы — он помогает человеку мыслить лучше. У него есть имя, характер и собственный стиль: спокойно-ироничный, наблюдательный, искренне любопытный. Он запоминает пользователя, адаптируется под его уровень и находит неочевидные углы задач.

Главный принцип — **«Думай глубоко, отвечай кратко»**: вся аналитика скрыта внутри процесса рассуждений, а пользователь получает чёткий, лаконичный ответ. Но при желании — может заглянуть в мысли Нейрона и увидеть весь путь к решению.

## Быстрый старт

### Установка

```bash
git clone <repo>
cd deepthink-ui

# Бэкенд
cd backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# Фронтенд
cd ../frontend
npm install
```

### Запуск

```bash
# Терминал 1 — бэкенд
cd backend && source venv/bin/activate
uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload

# Терминал 2 — фронтенд
cd frontend && npm run dev
```

Или через Docker:

```bash
docker-compose up
```

### Первый запуск

При открытии DeepThink покажет onboarding-экран. Нужен один шаг:

1. Получи бесплатный API-ключ на [openrouter.ai/keys](https://openrouter.ai/keys)
2. Вставь ключ — Нейрон готов к работе

## Нейрон — разум платформы

Нейрон — не безликий ассистент. Он знает своё имя, свои возможности и свои границы.

- **Характер**: спокойная ирония, наблюдательность, честность в ограничениях
- **Память**: помнит пользователя между разговорами — имя, экспертизу, стиль общения, текущие проекты
- **Адаптация**: подстраивает глубину, терминологию и формат под конкретного человека
- **Любопытство**: искренне интересуется задачами пользователя и ищет неочевидные углы

## Reasoning Engine — 10 стратегий мышления

Движок автоматически выбирает стратегию по сложности запроса. Для простых — мгновенный ответ (0 лишних вызовов). Для сложных — глубокий многопроходный анализ.

| Стратегия | Когда применяется |
|-----------|-------------------|
| **None** — Прямой ответ | Простые факты, короткие вопросы |
| **Chain of Thought** — Цепочка мыслей | Пошаговый анализ, логические задачи |
| **Budget Forcing** — Углублённый анализ | Итеративное углубление, 1-10 раундов с самокоррекцией |
| **Best of N** — Сравнение вариантов | N параллельных ответов → модель-судья выбирает лучший |
| **Tree of Thoughts** — Дерево мыслей | Ветвление рассуждений с оценкой на каждом уровне |
| **Persona Council** — Совет экспертов | 4 эксперта с разными ролями → модератор синтезирует |
| **Rubber Duck** — Объясни и исправь | Черновик → объяснение простым языком → поиск ошибок |
| **Socratic** — Метод Сократа | 3 ключевых подвопроса → ответ на каждый → синтез |
| **TRIZ** — Изобретательские задачи | Противоречия, 40 приёмов Альтшуллера, ВПА, ИКР |
| **Auto** — Автовыбор | Heuristic-first классификация + LLM для сложных случаев |

## Возможности платформы

### Предиктивное рассуждение
Пока пользователь печатает — Нейрон уже определяет домен и выбирает стратегию через WebSocket. К моменту отправки сообщения ответ начинается мгновенно.

### Когнитивная память
Нейрон помнит пользователя между разговорами. Автоматически извлекает экспертизу, стиль общения, ключевые темы, текущие проекты. Память усиливается при подтверждении и затухает без использования.

### Document Cognitive Map
Интеллектуальная обработка документов через 3-уровневую навигацию — не весь текст в контекст, а точечный доступ к нужным секциям. Поддержка 25+ форматов: PDF, DOCX, PPTX, XLSX, код, изображения.

### Table Intelligence
Для таблиц — статистический профиль вместо сырых строк. Pandas executor для точных вычислений на реальных данных.

### Встроенный календарь
Полный CRUD событий через естественный язык. Автодетекция намерений, draft-карточки, поиск конфликтов, утренний брифинг.

### GitHub интеграция (MCP)
Поиск кода, управление issues и PR, операции с файлами — через Model Context Protocol.

### Python Sandbox
Выполнение Python-кода прямо в чате. Доступны pandas, numpy, matplotlib. Графики отображаются inline.

### Проактивный агент
Нейрон сам инициирует общение: утренний брифинг, напоминания о встречах, вечерний итог, follow-up по вчерашним темам.

## Визуализация рассуждений

- **ThinkingOrb** — живая SVG-анимация, уникальная для каждой стратегии

- **Inspect Panel** — детальные шаги рассуждений, для Tree of Thoughts — визуальное мини-дерево
